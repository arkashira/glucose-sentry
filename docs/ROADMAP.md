# Roadmap - glucose-sentry

## Overview
glucose-sentry is developing a revolutionary non-invasive blood sugar monitoring device that uses AI-powered sensors to detect glucose levels without requiring finger pricks or blood samples. This roadmap outlines our development phases from MVP through v2.

## MVP (Minimum Viable Product) - Q3 2026

### MVP-Critical Features
- **Core Sensor Technology**: Implementation of initial non-invasive sensor prototype capable of detecting glucose levels through skin
- **AI Model Integration**: Basic glucose level prediction model trained on initial dataset
